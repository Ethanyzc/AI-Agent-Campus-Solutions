AI-Agent-Campus-Solutions
A Collection of Specialized AI Agents & Workflows Built on Dify

This repository showcases a series of AI-driven solutions designed to solve complex cross-border, financial, and procedural challenges. As a Computer Engineering graduate with a focus on AI Product Management, I developed these agents using First Principles to bridge the gap between LLM capabilities and real-world efficiency.

🚀 Featured Projects
1. Nvidia-CFO-Bot 📈
Role: Financial Analyst & Strategic Decision Support

The Problem: Financial reports and market data for tech giants like Nvidia are overwhelming for individual investors/analysts to process in real-time.

The Solution: A RAG-based (Retrieval-Augmented Generation) agent that ingests quarterly earnings and market sentiment to provide structured CFO-level insights.

Core Logic: Uses a multi-step workflow to verify data points against uploaded knowledge bases, reducing "hallucinations" in financial forecasting.

2. Swiss Legal/Financial Representative (cfo 瑞士) 🇨🇭
Role: Cross-border Compliance & Legal Advisory

The Problem: Navigating Swiss legal and financial regulations is high-cost and time-consuming for international entities.

The Solution: A specialized agent trained on Swiss regulatory frameworks to provide instant guidance on compliance, taxation, and legal representation.

Key Feature: Implements Chain-of-Thought (CoT) prompting to ensure the agent follows strict legal logic before providing an answer.

3. Advanced Dify Workflow Architecture (workflow) ⚙️
Focus: Efficiency & Modular Design

Concept: A reusable, modular workflow template designed to handle complex user intents.

Innovation: Integrates Intent Classification nodes and Conditional Branching to ensure the LLM handles edge cases (e.g., ambiguous queries) gracefully.

Application: Can be pivoted to any SaaS agent scenario, from customer support to automated PRD generation.

🛠 Tech Stack & Methodologies
Platform: Dify (Self-hosted/Cloud)

Frameworks: RAG (Retrieval-Augmented Generation), Agentic Workflows

Prompt Engineering: Few-shot prompting, Structured Output (JSON/Markdown), Role-play optimization.

Product Sense: User Journey Mapping, First Principles Thinking.
